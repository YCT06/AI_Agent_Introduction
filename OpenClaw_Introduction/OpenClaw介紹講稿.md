## 第 2 頁｜什麼是 OpenClaw？

**（解釋核心概念）**

那到底什麼是 OpenClaw 呢？

先來說說你們可能熟悉的 AI，像是 ChatGPT 或是 Siri。你問它問題，它回答你，對吧？但如果你說「幫我去網路上搜尋今天的天氣」，它頂多告訴你怎麼做，它沒辦法**真的幫你去做**。

OpenClaw 就不一樣了。它有「眼睛」，可以瀏覽網路；它有「手」，可以幫你操作電腦、讀寫文件、甚至發送訊息給別人。

而且更厲害的是，你不用坐在電腦前面操作——你只要拿起手機，在 LINE 或 Telegram 上打幾個字，它就去幫你做了。

用一個比喻來說：**OpenClaw 就像是你的超強私人助理，24 小時隨時待命，而且完全不需要付薪水**，因為它是免費的！

## 第 3 頁｜OpenClaw 的由來

**（說個故事，讓內容更生動）**

接下來我們來說說 OpenClaw 的由來，這個故事其實蠻有趣的。

OpenClaw 是一位叫做 **Peter Steinberger** 的工程師做的，他來自奧地利。在 2025 年 11 月，他把這個程式放到了 GitHub 上面——GitHub 就是一個工程師們分享程式碼的地方，就像是程式設計師的 YouTube。

那時候這個程式叫做「**Clawdbot**」。

結果，沒想到非常多人喜歡！但後來因為商標的問題，他在 2026 年 1 月改名為「**Moltbot**」。但這個名字念起來怪怪的，所以三天後，他又改名叫「**OpenClaw**」，也就是我們現在看到的名字。

更神奇的是，到了 2026 年 3 月，這個專案已經有超過 **68,000 個人在 GitHub 上按星星**——這個數量，代表著全世界非常多的工程師都覺得這個東西很厲害、很有用。

## 第 4 頁｜主要功能和特點

**（介紹六大功能，可以用手指著畫面一個一個講）**

好，接下來我們來看看 OpenClaw 到底有什麼功能。我幫大家整理成六個重點：

第一，**瀏覽網路**。它可以自動上網幫你查任何資料，填表單，甚至搜尋特定內容。

第二，**讀寫文件**。它可以讀你電腦裡的文件，也可以幫你寫內容進去，就像一個助手坐在你旁邊幫你整理資料。

第三，**多平台操控**。你可以用 Telegram、Discord、WhatsApp 來下指令——就是用訊息軟體跟它說話，它就去幫你做。

第四，**50 種以上的整合**。它可以連接超過 50 種不同的服務，包括 Notion 記事本、GitHub、甚至是智慧家電。

第五，**本地運行**。這一點很特別——它的資料全部存在你自己的電腦裡，不會上傳到別人的伺服器，所以隱私比較有保障。

第六，**記憶功能**。它記得你之前說過的事情，越用越了解你的習慣，就像認識你很久的老朋友一樣。

## 第 5 頁｜如何運作的？

**（用簡單的四步驟流程圖說明）**

那它到底是怎麼運作的呢？我用最簡單的方式來解釋，分成四個步驟：

**第一步：你發出指令。**
你用 Telegram 或是其他訊息軟體，用日常語言打一句話，比如說「幫我查台北今天的天氣」。

**第二步：AI 理解你的意思。**
OpenClaw 背後連接著大型 AI 模型——你可以把它想像成一個超聰明的大腦——這個大腦看完你說的話之後，會想好要怎麼做。

**第三步：執行動作。**
它真的去上網查資料，或者讀你的文件，或者發通知給你——就是真的去「做」這件事。

**第四步：回報結果。**
做完之後，它會傳訊息告訴你：「我已經幫你查好了，台北今天 25 度，晴天。」

這四個步驟，全程幾秒鐘就完成了，完全不需要你手動操作。

另外補充一點——OpenClaw 支援多種 AI 大腦，包括 Claude、GPT-4、DeepSeek，你可以自己選擇要用哪一個。

設定好之後，就可以直接打訊息給它了。比如說「幫我查今天台北天氣」，它就去做了。

舉個例子：你可以說「幫我把今天的會議記錄整理成表格，並存到 Notion」——它就真的去做這件事，不需要你動手！

## 第 10 頁｜安全性和隱私保護措施

**（坦誠說明優點與注意事項）**

說到這裡，也許有些同學會擔心：「這東西那麼強大，安全嗎？會不會有隱私問題？」

這個問題問得很好！我來說明 OpenClaw 在安全和隱私方面的設計：

**本地資料儲存**——所有的設定和對話記錄都存在你自己的電腦裡，不會上傳到別人的伺服器。你的資料，你自己管。

**自備 API 金鑰**——你使用自己的 AI API 金鑰，連 Anthropic 或 OpenAI 這些公司都看不到你的使用內容。

**開放原始碼**——程式碼完全公開，任何工程師都可以去檢查，確保沒有偷偷做壞事、沒有後門或惡意程式。

當然，也有需要注意的地方：

OpenClaw 需要比較大的系統權限才能運作，所以在設定的時候要特別小心，不要亂授權給陌生人。另外，如果設定不當，可能讓別人能夠透過 Bot 存取你的電腦，所以務必要設定好存取密碼。

總結來說，只要設定正確，OpenClaw 是相對安全的工具。

## 備用問答參考

**Q：OpenClaw 和 ChatGPT 有什麼不同？**
A：ChatGPT 只能對話、給你建議；OpenClaw 可以真正幫你「做事」，例如上網查資料、讀寫文件、發送訊息等。

**Q：要花錢嗎？**
A：OpenClaw 本身完全免費。不過使用 AI 功能（如 ChatGPT 或 Claude 的 API）可能需要一點費用，但也有完全免費的 AI 模型可以選擇。

**Q：不會程式可以用嗎？**
A：目前需要一些基本的電腦操作能力來安裝。但開發團隊的近期目標之一，就是推出更簡單的圖形化介面，讓完全不懂程式的人也可以使用。

**Q：安全嗎？會不會被駭客攻擊？**
A：只要設定正確、設好密碼，是相對安全的。因為資料存在自己電腦，比存在別人伺服器更安全。但任何工具都需要謹慎使用。

**Q：台灣可以用嗎？**
A：可以！它支援全球使用者，中文語系也在開發中。目前已有台灣使用者在使用。
